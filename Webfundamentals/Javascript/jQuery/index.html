jQuery Cheatsheet!

Show and Hide

.hide(); and .show(); will hide and show elements from the page.

Toggling

.toggle(); is essentially .hide(); and .show(;) in one method.

Sliding elements

.slideDown(); will display the elements with a sliding motion and .slideUp(); will hide elements with a sliding motion. Method name is case sensitive!

Slide Toggling

.slideToggle(); is essentially .slideUp(); and .slideDown(); in one method. Case sensitive!

Fading Elements

.fadeIn(); and .fadeOut(); will fade elements to/from the page.

Adding/removing Classes from elements

.addClass(); can add any class to an element and .removeClass(); can remove one, multiple, or all classes if specified.

Before and After

.before(); will insert content before the specified element and .after(); will place it after the element.

HTML method

.html(); will display the actual HTML of the target element.
